"p" tag.

A "p" tag serve para criar um parágrafo.

Exemplo: Lorem ipsum, dolor sit amet consectetur adipisicing elit. Delectus laudantium amet minus necessitatibus harum inventore voluptates cum beatae at. Libero natus asperiores provident laboriosam dolore in error fuga, molestias expedita.

"br" tag.

A "br" tag serve para uma quebra de linha no texto.

"hr" tag.

Cria uma quebra de linha mas com uma linha horizontal na página.

Listas.

Temos dois tipos de listas, as ordenadas e a não-ordenadas.

Lista ordenada.

  1. Laranja.
  2. Maça.
  3. Pêra.

Lista não-ordenada.

  • Aluno.
  • Professor.
  • Líder de turma.

Imagens.

Para colocarmos uma imagem em um site, utiliza-se a tag "img", onde o atributo "src" é o a chamada da imagem, e o atributo "alt" é o nome dado a essa imagem em seu site.

Links.

Servem para redirecionar o usuário para outra página, utiliza-se a tag "a"
